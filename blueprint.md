# Blueprint: Jasper AdSense Test

## Overview
This project is a framework-less web application designed to test Google AdSense integration. It is built using modern web standards (HTML, CSS, JavaScript) and will be deployed to Cloudflare Pages.

- **Deployment URL:** [https://jasper-adsense-test.pages.dev](https://jasper-adsense-test.pages.dev)
- **Framework:** Vanilla JavaScript (ES Modules)
- **Styling:** Modern CSS (Baseline)
- **Components:** Web Components (Custom Elements)

## Project Outline
- `index.html`: Main entry point.
- `style.css`: Global styles using modern CSS features.
- `main.js`: Main JavaScript logic using ES Modules and Web Components.
- `.idx/`: Project environment configuration.

## Current Plan: UI Modernization & AdSense Integration
1. **Modernize UI:** Update `index.html`, `style.css`, and `main.js` to create a polished, responsive, and modern look.
2. **Web Components:** Use Web Components for encapsulated UI elements.
3. **AdSense Integration:** Add placeholders for AdSense advertisements.
4. **Deployment:** Push changes to GitHub `origin main` to trigger the production deployment to `jasper-adsense-test.pages.dev`.

## Design Goals
- **Aesthetics:** Clean, modern layout with ample whitespace.
- **Typography:** Using high-quality system fonts or Google Fonts.
- **Interactivity:** Subtle hover effects and smooth transitions.
- **Responsiveness:** Fully functional on mobile and desktop.
